In [112]:
def get_mean(nums):
    if(len(nums) == 0):
        return 0
    else:
        total = 0
        for num in nums:
            total += num
        return round(total / len(nums), 2)

Write and test another function

Let's write a function that returns the standard devation of a list of numbers, rounded to the nearest hundreth decimal place.

In [117]:
def get_sd(nums):
    mean = get_mean(nums)
    total = 0
    for num in nums:
        total += (num - mean) ** 2
    variance = total / (len(nums) - 1)
    return round(variance ** 0.5, 2)

Find Pearson correlation coefficient (r) between two variables

It's easy with numpy.

In [122]:
correlation = np.corrcoef(d_data, s_data)
print(correlation)

[[ 1.         0.0152266]
 [ 0.0152266  1.       ]]

Uh... what is this?
Well that's actually a matrix containing all combinations of r coefficients for the two datasets entered.
Half of them are identical since each dataset is compared to the other, as well as itself (yielding an r of 1.0).
Here is an image to give you a better idea.

Let's access one of those valid r coefficients and round it off to two decimal places.

In [123]:
correlation = round(np.corrcoef(d_data, s_data)[0][1], 2)
print("Pearson's r =", correlation)

Pearson's r = 0.02


Our r coefficient is about 0.02. This means the relationship is pretty much nonexistant.
Or more properly, we'd say defense does not predict speed (and vice versa).
My hunch was wrong!

Create a model

Now that we have identified at least a moderate relationship between two variables, we can think about regression.
The scatter plot we made appears to follow a linear (y = ax + b) model, doesn't it?
So let's make one with numpy and see how it fits.

In [126]:
model = np.polyfit(hp_data, a_data, deg=1)
print(model)

[  0.53689898  41.81629745]

np.polyfit() returns a list-like structure containing the coefficients and constants of an equation, based on what we entered for "deg" or degree.
We anticipate a linear or "first degree" equation, so we entered "1" as the value for "deg". More on equation degrees here.
So our regression equation would look like this:
y = 0.54x + 41.82
